In Assignnment_3 we have two simple Python programs file Task_1.py & Task_2.py created to demonstrate below Python concepts 

Task_1.py – Calculate Factorial Using a Function

This program:
Takes a number as input from the user
Defines a function named factorial that takes a number as an argument and calculates its factorial using recursion.
Returns the calculated factorial.
Calls the function with a user given number and prints the output.

Task_2.py – Using the Math Module for Calculations

This program:
Takes a number as input from the user
Using the math module by importing it to calculate the:
	a)Square root of the number
	b)Natural logarithm (log base e) of the number
	c)Sine of the number (in radians)
Calls the function with a user given number and prints the output.
